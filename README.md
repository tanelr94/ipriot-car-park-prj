# Moondalup Car Park System
## Overview
This is a prototype smart car park system for the City of Moondalup, implementing real-time parking bay tracking, sensor detection, and display updates using OOP principles.

## Features
- Tracks available parking bays
- Records car license plates
- Displays real-time status
- Includes unit tests

## Setup
1. Clone the repository
2. Create and activate a virtual environment
3. Run `main.py`# ipriot-car-park-prj
